api

the api is really simple and only has a few codes, and one route

Codes

Code Meaning
0 Unauthorized
1 Okay
2 No emails
3 Not found
4 Generic server error

GET /:inbox

Headers

Header Value
Authorization (your authorization key, if enabled)

Example responses:

200 Success

{
    "data": {
        "inbox": "neumatic",
        "subject": "hello world",
        "from": "neumaticc@gmail.com",
        "to": "neumatic@neumatic.club",
        "html": "<h1>based email server</h1>",
        "text": "based email server"
    },
    "code": 2
}

400 Error: inbox is empty

{
    "error": "no emails",
    "code": 2
}

401 Error: authorization token missing or invalid

{
    "error": "unauthorized",
    "code": 0
}
